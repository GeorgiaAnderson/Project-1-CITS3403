Overview of CSS

Purpose and Motivation of CSS

CSS is a language used for describing the presentation of web pages. Its independent of HTML, and it allows developers to control the appearance and display of their website because its based on a set of formatting rules. These rules provide a standard way to define, apply and manage style characteristics like colour, layout, and font and also provide a way to make web pages responsive to different screen sizes and device types.

Before CSS there was very little that could be done to change the style of a web page. As more people started to use HTML the demand for styling capabilities grew but HTML was designed just to hold content and was not capable of complex styling. In 1994 Hakon Wium released the first draft of CSS but it wasn't until 1996 that W3C released its first recommendation. Since its implementation by browsers web pages have become far more functional, have improved design, have some consistency and have reduced the amount of bandwidth needed to load them.

Benefits of seperating content from style

  • Less code -- By using a CSS style sheet that can be applied to multiple pages rather than styling each individual item in a html document, you reduce the amount of code your web page uses which shortens web page loading times.
  • Bandwidth reduction -- By separating content from style you decrease the file transfer size because the CSS document will be stored externally and will only be examined when a visitor requests your website .
  • Maintainability -- Using CSS style sheet makes maintaining and updating websites styles easy because you only need to update the style for an element once, CSS will automatically change the style of that element on every page throughout the website.
  • Consistency -- By applying one style sheet to your web page rather than individually styling each element you increase the consistency of your styling and save yourself a lot of time.

Key CSS Selectors

Cascading property

The cascade property in CSS is an algorithm that decides which CSS style will apply to an element. The algorithm takes into account 4 factors of deceasing importance, the first being what style sheet the declaration is made in. Declarations in author style sheets will always override those of user style sheets except in the case of important declarations set by the user, these will override all other declarations. The second factor is importance, a declaration with the “!important” tag will override all other declarations. The third factor is specificity, this is a number that the browser calculates when applying css and is based on the types of selectors the declaration uses. The forth factor is source order, where if you have more than one declaration with the same weight then the one that comes last in the css code will be applied.

In my example i demonstrate the effect that the cascade has on the competing background colour of paragraphs. It shows two things: (1) Even though the cascadeDiv is set to blue the paragraphs inside the div who who have a class or ID are not blue because their background colour declaration is more specific. (2) ID is more specific than class and hence the paragraph with an ID is set to pink, overwriting the blue and green colours.

CSS

.cascadeDiv {
background-color: #cfe0e8;
}

.cascadePara {
background-color: #cfffe5;
}

#cascadeParaId {
background-color: #fadadd;
}

HTML

<div class = "cascadeDiv">

<p> this is a paragraph with no attributes </p>

<p class = "cascadePara">this is a paragraph inside div with a class </p>

<p class = "cascadePara" id="cascadeParaId">this is a paragraph inside div with an id and class </p>

</div>

<p class = "cascadePara">this is a paragraph outside the div with a class </p>

Output

this is a paragraph inside div with a class

this is a paragraph inside div with an id and class

this is a paragraph outside the div with a class

Selector Definition

Selectors define which HTML elements are going to be manipulated by CSS code.

Basic Selectors

Universal selector: Is denoted by an "*" and is used to select any type of element in an HTML page.

/*This CSS will select all elements and change their colour to green*/

* {
color:green;
}

Class selector: Denoted by a "." and is used to select HTML elements with a specific class attribute

/*This CSS will select all HTML elements with the class "classSelec" and change their colour to blue*/

.classSelec {
color:blue;
}

ID selector: Denoted by "#" and is used to select HTML elements with a specific id attribute

/*This CSS will select all HTML elements with the id "idSelec" and change their background colour to pink*/

#idSelec {
background-colour:pink;
}

Attribute selector: Denoted by "[ ]" and is used to select HTML elements with a specific attributes.

/*This CSS will select all HTML elements with an anchor tag and href attribute and make them orange*/

a[href] {
color:orange;
}

Child selector: Denoted by ">" it is placed between 2 css selectors and selects HTML elements that match the second selector and that are direct children of the first selector

/*This CSS will select all h4 elements that are inside a div and make their background blue*/

div > h4 {
backround-color:blue;
}

Pseudo Selectors

Pseudo Classes: Are a selector that selects elements in a specific state e.g. in a particular position relative to another element. They allow you to apply css styles to elements without having to give them a class or id name, giving you more flexible and maintainable code.

First-Child pseudo class: Used to select the specified selector only if its the first child of its parent

/*This CSS will select the paragraph elements and change them to green if they are the first child of their parent*/

p:first-child {
color:green;
}

Hover pseudo class Used to select elements as you move your mouse over them

/*As you hover over the position buttons in the interactive example this CSS with differing position attributes is applied*/

.posAbs:hover {
position:absolute;
}

Pseudo Elements Denoted by "::", they represent entities that are not included in HTML and used to style specific parts of an element.

First-Letter pseudo element Used to select the first letter of the first line of a block level element.

/*This CSS will select the first letter in the first line of the h4 heading and make it orange*/

h4::first-letter {
color:orange;
font-size:130%
}

After pseudo element Creates a pseudo-element thats the last child of the selected element

/*This CSS will create a new element after h5 and add content that says "pseudo elements are cool" and colour the text blue*/

h5::after {
content:"pseudo elements are cool";
color: cornflowerblue;
}
